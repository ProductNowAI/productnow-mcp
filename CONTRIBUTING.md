# Contributing

This repository is maintained by ProductNow as the public metadata and
documentation source for the hosted ProductNow MCP server.

Before changing listing metadata:

1. Update the relevant JSON file.
2. Update the matching documentation page.
3. Validate that the production MCP endpoint and OAuth discovery URLs still
   match the published metadata.
4. If the tool surface changed, update both `tools.json` and `docs/TOOLS.md`.

Do not add ProductNow application source code, credentials, customer data, or
private staging metadata to this repository.
